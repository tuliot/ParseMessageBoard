ParseMessageBoard
=================

Parse-enabled iOS messaging view kit.

To use this, please add the following frameworks to your project.

  -ParseUI
  -Parse
  -Bolts
  -MobileCoreServices
  -SystemConfiguration
  -StoreKit
  -Security
  -libz.1.1.3.dylib
  -QuartzCore
  -AudioToolbox
  -CFNetwork
  -CoreGraphics
  -CoreLocation
